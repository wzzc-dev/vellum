# Vellum MoonBit SDK extension examples

Each directory in this folder is a standalone MoonBit extension project that
builds to a WASM Component for Vellum's `vellum:extension/extension-world` host.

Legacy week 1-6 snippets that used `@gui.*` have been replaced by the equivalent
extension directories in this folder.

Build all examples:

```sh
./build-all.sh
```

Build one example:

```sh
cd counter
./build.sh
```
